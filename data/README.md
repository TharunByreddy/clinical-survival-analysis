# Data Directory

## About
This folder contains clinical patient data used for survival analysis.

## How to populate
Data is automatically generated by running:
```bash
python src/preprocess.py
```

Or bring your own clinical dataset as a CSV with these columns:

| Column | Description |
|--------|-------------|
| `patient_id` | Unique patient identifier |
| `age` | Patient age |
| `gender` | Male / Female |
| `diagnosis` | Diagnosis type (Type A, B, C) |
| `comorbidities` | Number of comorbid conditions |
| `treatment` | Treatment received |
| `duration` | Survival duration in days |
| `event_observed` | 1 = event occurred, 0 = censored |

## Data Source Inspiration
Based on real-world clinical registry data (DHS/SRTR) used in
transplant outcome research at Stanford University School of Medicine.
